Scrambled Word Game

Java | 2017/18

My first-ever independent project using a "real" programming language was this Scrambled Word Game that I designed myself and programmed in Java during my sophomore year of high school. The game has multiple difficulty levels: easy, medium, and hard. The player has a three chances to correctly guess a randomly-selected word that has been scrambled using a different shuffle algorithm, depending on the selected game level.
