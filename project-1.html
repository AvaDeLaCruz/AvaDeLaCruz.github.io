AAPI Governance Body Meeting Scheduling

HTML5, CSS, JavaScript, jQuery, Node.js, AWS Lambda, DynamoDB, Simple Email Service, and API Gateway | May-August 2019

During my Summer 2019 internship at Amazon HQ in Seattle, WA, I had the incredible experience of learning full-stack development as well as learning how to use numerous AWS services. Over the course of twelve weeks, I worked on creating a meeting scheduling website for data providers to use when choosing a date and time for their proposal review meeting with the AmazonAPI Governance Body, the group which approves changes to API definitions which affect Amazon.com. My project is now being used by 20-25 data providers each week. By automating user authentication, user authorization, time slot selection, time slot reservation, calendar event creation, and attendee invitation via email notifications, I was able to successfully reduce the time required to schedule a meeting by 80% on average.
